Billing Model
Fixed (always-on 24/7): ML $61,797/mo + MC $3,256/mo + MP $690/mo = $65,743/mo baseline
Variable (per match): CloudFront GB delivery + ML $736/match + MC $19.26/match + MP tier cost
CDN costs: Enter ME/EU GB from AWS Cost Explorer for exact figures, or auto-estimate via bitrate × viewers × watch time
4K add-on: CH11_SPL only (+$8.244/hr). CH10_SPL is Reserved/Excluded
⚠ Scope: This model covers AWS media services only (CloudFront, MediaConnect, MediaLive, MediaPackage). Alibaba CDN and other delivery partners are not yet included. Will recalibrate each season as new data becomes available.
